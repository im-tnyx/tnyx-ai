# TNYX Mobile Accessibility Guidelines

## Purpose

Ensure usable UI for all users across Android and iOS.

## Mandatory Rules

- meaningful semantics/labels for interactive elements
- sufficient color contrast
- touch targets should remain comfortably tappable
- support dynamic text scaling without breaking layout
- avoid motion that harms readability or usability

## Interaction Guidance

- primary actions must be clearly discoverable
- do not rely on color alone to convey meaning
- loading/error messages should be understandable and actionable

## Validation Checklist

- screen-reader labels present for buttons/icons
- focus order remains logical
- important state changes are perceivable
